Data Projects

I’ve worked on projects across industries like finance, health, education, and operations — building dashboards, performance models, and interactive reports using tools like Microsoft Power BI, SQL, and Microsoft Excel.

Uber Ride Analytics Dashboard

This project analyzed Uber ride operations using Excel and Power BI to uncover trends in rider behavior and demand. Results showed high night-time (39.6%) and weekend activity, strong demand in the Upper East and Upper West Side, and a cash-dominant payment pattern (58.59%). The insights guided recommendations for better driver deployment, surge pricing optimization, and improved operational efficiency.

Hospital Emergency Room Operations Dashboard

This healthcare analytics project examined 9,128 emergency room visits (April 2023–October 2024) using Excel and Power BI to assess patient flow, wait times, satisfaction, and resource use. Average wait time was 35.3 minutes, with 59.32% meeting the 30-minute target and a satisfaction score of 4.99/5. Insights supported recommendations to optimize staffing, enhance triage, and improve referrals to reduce wait times by 20%.

Bank Loan Portfolio Analysis Dashboard

This financial analytics project evaluated 38,600 loan applications totaling $435.8M funded. Using Power BI and Excel, it assessed lending trends, risk profiles, and portfolio health. Results showed 86.2% good loans, steady monthly growth ($25M–$54M), debt consolidation as the top purpose, and 10+ year employees as major borrowers. Insights guided strategies to cut bad loans below 10%, refine pricing, and apply predictive default models.
